# github-release-pipeline

Claude Code skill for cross-platform build and release automation using gh CLI and GitHub Actions.

## What it does

- gh CLI operations for repo, release, and workflow management
- Windows/macOS/Linux matrix build workflow template
- Step-by-step release publishing flow
- 5 common issues with root cause analysis and fixes
- 7 best practices

## Installation

```bash
git clone https://github.com/cyforkk/github-release-pipeline.git
cp github-release-pipeline/SKILL.en.md your-project/skills/github-release-pipeline/
```

## Usage

After installation, tell Claude Code:

> Use the github-release-pipeline skill

## Requirements

- gh CLI installed (`brew install gh` / `winget install GitHub.cli`)
- GitHub account authenticated (`gh auth login`)
